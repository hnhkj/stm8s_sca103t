## SCA103T

一款单轴加速度传感器，具有模拟输出和数字输出两种接口。可以提供精确的单轴角度信息。
适合用于角度测量的一些场合。


## STM8S003

该设计采用了低容量低价格的STM8S003系列单片机，该单片机具有极低的价格和强大的性能。
是一款性价比非常高的单片机。

## 实现功能

该设计通过了SPI接口读取SCA103T的内部两路传感器信息和一路温度信息，通过单片机的串口
输出温度，ST1, ST2，ST1-ST2, ST1+ST2信息。从而可以进行SCA103T的数据检测和测试。
